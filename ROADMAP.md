# Roadmap

- improve schema-aware token heuristics
- add side-by-side comparison for multiple manifests
- add more sample manifests from real servers
- import MCP Inspector exports directly in the browser UI

- runtime proxy mode that enforces selected packs

- client-specific config generators for more SDKs
- diff view between the full toolset and the selected pack
- saved analysis reports

## Longer term

- live MCP server introspection
- policy packs for common deployment environments
- collaboration review mode for team approvals
